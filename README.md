# Webpack Starter Kit

A starter kit [Boiler Plate] that i intend to reuse for some of my future ES6 & Babel projects.

## Dependencies

+ webpack@3.10.0
+ babel-polyfill@6.26.0
+ babel-loader@7.1.2
+ babel-preset-es2015@6.24.1
+ webpack-dev-server@2.9.7
+ babel-core@6.26.0
